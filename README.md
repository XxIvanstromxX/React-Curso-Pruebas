# Curso básico de React (React-Curso-Pruebas)

Bienvenido: este repositorio contiene un curso práctico y incremental de React con TypeScript y Vite. Iré agregando una carpeta por cada clase (por ejemplo `Clase1`, `Clase2`, ...) y mantendré en cada una una pequeña app o ejemplo que se pueda ejecutar y estudiar.

Resumen rápido
- Stack: React + TypeScript + Vite (plantilla mínima por clase).
- Estructura: cada carpeta `ClaseN/` contiene su propio proyecto (package.json, src, README, etc.).

Cómo usar una clase
1. Entra en la carpeta de la clase que quieras (por ejemplo `Clase1`).
2. Revisa su `package.json` para ver los scripts disponibles. Normalmente encontrarás scripts como `dev`, `build`, `preview`, `lint`.
3. Ejecuta el script que necesites con tu gestor de paquetes (por ejemplo `pnpm`, `npm` o `yarn`) según tu preferencia.
