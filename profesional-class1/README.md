# React Intermedio - Clase 1

Este es un repositorio de prueba para la **Clase 1 de React Intermedio**.

Curso creado por **Ivan Martinez** para **CCOL**.

## Proyecto del Curso

Durante las siguientes **6 clases** construiremos un **CRM (Customer Relationship Management)** completo utilizando React.

## Clase 1: Estructura del Proyecto

En esta primera clase nos enfocaremos en:

- **Estructura del proyecto**: Organización de carpetas y archivos
- **Carpetas principales**: Definición y configuración de directorios clave
- **Configuración base**: Setup inicial del proyecto con las herramientas necesarias
- **Dependencias**: Instalación de librerías esenciales para el desarrollo
